# 개발자 웹 이력서 프로젝트 로드맵

## 프로젝트 개요

개인 개발자 웹 이력서 사이트를 구축하는 프로젝트입니다.

## 기술 스택

| 구분 | 기술 |
|------|------|
| 마크업 | HTML5 |
| 스타일링 | CSS3, TailwindCSS |
| 인터랙션 | JavaScript (Vanilla) |

## 프로젝트 구조

```
/
├── index.html          # 메인 이력서 페이지
├── css/
│   └── style.css       # 커스텀 스타일
├── js/
│   └── main.js         # 인터랙션 스크립트
└── assets/
    └── images/         # 프로필 사진 등 이미지 리소스
```
